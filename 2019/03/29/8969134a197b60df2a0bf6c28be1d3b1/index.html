﻿# 1、流程
1、流程如下：解压.7z压缩包，解压完之后图片都在dir(自己命名)文件夹下；
2、然后执行 ls dir >file_list.txt(文件名自己命名)，将图片名称列表存放到txt文件中；
3、然后执行sh form_filename.sh file_list.txt dir

2、sh文件form_filename.sh

新建一个文本文档，将下面复制到里面，另存选择所以文件，命名为.sh 结尾就行。然后按流程即可。

#!/bin/bash
file=$1
dir=$2
cat $file | while read line
do
  old_picname=$line
  new_picname=`echo $old_picname|sed 's/,//g'|sed 's/=//g'|sed 's/&//g'|sed 's/%//g'|sed 's/!//g' \
  |sed 's/(//g' | sed 's/)//g'`
  echo $new_picname
  mv ${dir}"/"$old_picname ${dir}"/"$new_picname
done
