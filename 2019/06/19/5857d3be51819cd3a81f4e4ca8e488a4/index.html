  • 其他

@[toc]

1、截全屏

1、Ctrl + shift + I 进入开发者工具（也可以右上角三个点，选择更多工具，然后点击开发者工具）
2、Ctrl + shift + P,然后输入srceen，然后选择capture full size screenshot
3、截图保存下来，在电脑的下载文件夹

2、截某个元素，也就是截图一分部

1、Ctrl + shift + I 进入开发者工具（也可以右上角三个点，选择更多工具，然后点击开发者工具）
2、在elements 里面找到需要截图的div或者其他元素块，通过移动颜色加深的就是选择的元素块。

2、Ctrl + shift + P,然后输入srceen，然后选择capture full node screenshot

3、截图保存下来，在电脑的下载文件夹

4、注意：有个缺点，只能截图窗口显示部分，其实这个功能还不如用qq,微信，钉钉等聊天工具软件的截图方便。
