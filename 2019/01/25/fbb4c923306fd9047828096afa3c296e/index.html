﻿# 使用协程爬虫monkey.patch_all()
运行报异常：

MonkeyPatchWarning: Monkey-patching ssl after ssl has already been imported may lead to errors
{% image /imgs/20190125095509197.png '在这里插入图片描述' '' %}

这个上面图片是我报错时的导入模块顺序，将下面这三个放在最上面即可解决此问题。

import gevent
from gevent import monkey
monkey.patch_all()


顺更改之后，异常警告没有了。
