np.c_和np.r_

2019-12-02
  • python模块

﻿# demo
import numpy as np
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
c = np.c_[a,b]

print(np.r_[a,b])
print(c)
print(np.c_[b,c])

结果

[1 2 3 4 5 6]
[[1 4]
 [2 5]
 [3 6]]
[[4 1 4]
 [5 2 5]
 [6 3 6]]

np.r_是按列连接两个矩阵，就是把两矩阵上下相加，要求列数相等。
np.c_是按行连接两个矩阵，就是把两矩阵左右相加，要求行数相等。
